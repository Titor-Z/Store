# Store 我的店·系统

## 主要包含以下内容
- 小程序后台接口 (API接口)
- 用户后台系统 （我的店）
- 微格商铺管理系统（总平台）

## 说明

### 小程序后台接口
- 包含了 *我的店* 和 *总平台*中
    - 数据访问层
    - 文件（上传、查询、删除）操作
- 包括了总平台的 *逻辑处理*：
    - session（权限、登录、检测、访问）操作
- 小程序接口层

### 用户后台系统
- 包含了 *我的店*系统的完整处理

### 总平台
- 采用前后端分离，半自动化操作（容易出现前端逻辑异常的问题，因为没有采用mvvm框架操作）
- 只存在静态文件
